Evaluating Functions for a Given Input

  • Evaluating functions for a given input is the process of using an equation that states "y = x" or "f(x) = x" and solving that equation.

Slope Formula

  • Slope formula is the equation used to find the slope of a sequence of numbers

Slope-Intercept Form of an Equation (y = mx + b)

  • Slope-intercept form is the equation used to graph a linear line onto a coordinate plane. It is the same as the slope formula, but it includes the "b" in the equation, which moves the intercept location of the line on the y-axis

Standard Form of an Equation

  • Standard form is a form of a linear equation that allows you to find the x-intercept, y-intercept, and slope of an equation just by looking at it. It is structured as Ax + By = C

Key Characteristics of Linear Functions

  • Some key characteristics of linear functions is the slope (m), which determines how much the line moves on the y-axis per one unit on the x-axis, and the y-intercept (b), which determines at what point the line intersects with the y-axis

Arithmetic Sequences

  • An arithmetic sequence is a sequence of numbers that follows a constant progression.

  • Explicit Formula:

  • The explicit formula of an arithmetic sequence allows you to plug in any location of a number in the sequence to get the value of said location

  • Recursive Formula:

  • The recursive formula of an arithmetic sequence requires that you plug in the previous number to acquire the next number in the sequence
